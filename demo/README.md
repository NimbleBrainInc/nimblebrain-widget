# Demo Files

This directory contains test pages and demo implementations for the NimbleBrain widget.

## Usage

These files are for testing and demonstration purposes. The main test page used in production builds is `/index.html` in the root directory.

To test locally:

1. Build the widget: `npm run build`
2. Serve locally: `npm run serve`
3. Visit any demo file at `http://localhost:8080/demo/[filename].html`
